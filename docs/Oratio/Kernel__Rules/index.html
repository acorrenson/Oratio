Module Kernel__Rules

Rules

This module provides a backend for Oratio's engine to build valid by construction sequents.

The type thm represents sequents. Functions of this modules implements introduction and elimination rules as smart constructors of this type.

The Theorem type

type thm

The type representing valid sequents

type prop = Kernel.Logic.prop
val show_thm : thm -> string

Stringify a thm object for pretty printing purposes

val get_ctx : thm -> prop list

Get the context associated to a sequent. If t represents the sequent Γ ⊢ p then get_ctx t is Γ.

val get_prop : thm -> prop

Get the proposition associated to a sequent. If t represents the sequent Γ ⊢ p then get_ctx t is p.

val is_proof : thm -> prop -> bool

Check either a thm is a proof of a given proposition or not. is_proof thm prop is true if and only if get_ctx thm is the empty context and get_prop thm is prop.

The inference rules

Introduction and elimination rules for the natural deduction in propositional logic. Rules produce a thm object or fail displaying an error message.

val axiom : prop list -> prop -> thm

Axiom rule. axiom Γ p represents the sequent Γ ⊢ p where p is in Γ.

val intro_impl : thm -> prop -> thm

Implication Introduction. intro_impl (Γ α ⊢ β) α represents the sequent Γ ⊢ α → β.

val elim_impl : thm -> thm -> thm

Implication Elimination a.k.a Modus Ponens rule. elim_impl (Γ ⊢ α → β) (Γ ⊢ α) represents the sequent Γ ⊢ β.

val elim_bot : thm -> prop -> thm

Bottom Elimination a.k.a Ex Falso rule. elim_bot (Γ ⊢ ⊥) α represents the sequent Γ ⊢ α.

val intro_or_l : thm -> prop -> thm

Or Introduction (left) rule. intro_or_l (Γ ⊢ α) β represents the sequent Γ ⊢ α ∨ β.

val intro_or_r : thm -> prop -> thm

Or Introduction (right) rule. intro_or_l (Γ ⊢ α) β represents the sequent Γ ⊢ β ∨ α .

val elim_or : thm -> thm -> thm -> thm

Or Elimination rule. elim_or_l (Γ ⊢ α ∨ β) (Γ α ⊢ μ) (Γ β ⊢ μ) represents the sequent Γ ⊢ μ.

val intro_and : thm -> thm -> thm

And Introduction rule. intro_and (Γ ⊢ α) (Γ ⊢ β) represents the sequent Γ ⊢ α ∧ β.

val elim_and_l : thm -> thm

And Elimination (left) rule. elim_and_l (Γ ⊢ α ∧ β) represents the sequent Γ ⊢ α.

val elim_and_r : thm -> thm

And Elimination (right) rule. elim_and_r (Γ ⊢ α ∧ β) represents the sequent Γ ⊢ β.
