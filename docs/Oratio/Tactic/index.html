Module Tactic

Tactic

Tactics for interative theorem proving.

Each tactic is a function maping an interactive proof environement to another proof environement and generating instructions for the proof construction engine.

The program generated by succesive applications of tactics can then be evaluated using the engine module together with the backend of your choice.

The function qed evaluate the construction program contained in a proof environement with the rules backend for Oratio's engine.
